DIVS

The DiVS tag on its own doesn't do much.

DIV tag definition:

  • DIV tag has no meaning
  • DIV tag defines a division
  • HTML is not funny

DIV tag usage

Assigning an id to a tag allows us to style that tag (and all its nested tags) directed through CSS.

To give that id styling it must be named as a selector along with its declaration block, and preceded by a pound(hashtag symbol).

ID usage

The id allows us to identify tags so that they may specifically be addressed within the CSS file.
